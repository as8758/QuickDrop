# Icon Setup

The extension needs icon files for the Chrome extension. You have a few options:

## Option 1: Use Default Icons (Quick Start)
Chrome will use default icons if these files don't exist. The extension will work fine, but you won't have custom icons.

## Option 2: Create Simple Icons
Create three PNG files:
- `icons/icon16.png` - 16x16 pixels
- `icons/icon48.png` - 48x48 pixels  
- `icons/icon128.png` - 128x128 pixels

You can use any image editor or online tool to create simple icons with:
- A drag-and-drop symbol
- A form/document symbol
- Or just text "QD" for QuickDrop

## Option 3: Generate Icons Online
Use a tool like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Or any icon generator

Upload a simple logo/icon and download the PNG files.

For now, the extension will work without custom icons - Chrome will use defaults.
